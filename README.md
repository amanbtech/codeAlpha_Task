# codeAlpha_Task

## 🔢 Calculator Project

This is a custom-built calculator made **without using `eval()`** or shortcut hacks.

### ⚙️ Features:
- Built using **pure JavaScript**
- Uses `map()` method to handle operations
- No use of `eval()` — custom logic for safety and learning
- Simple and clean UI

### 🛠️ Tech Stack:
- HTML
- CSS
- JavaScript

### 🌐 Live Demo:
[Click here to try the calculator](https://amanbtech.github.io/codeAlpha_Task/)

### 📂 Folder Contents:
- `index.html` – Calculator layout
- `style.css` – Styling
- `script.js` – Logic using `map()`
